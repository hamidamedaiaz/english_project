# MOCK OF R&D PROJECT SEEKING INVESTORS: WATER FUELED ENGINES

## DESCRIPTION
As part of a class activity in English class for the 8th semester of engineering at Polytech Nice Sophia, we are given as task to design the landing page of a fictional project about a technology to soon become real world product as we are the creators of that product.

The aim is to make it believable and at the same time crazy enough for someone to "get the joke". As you can understand from the title, our fake product is engines that are fueled by water.

The inspiration comes from research works that was aimed at making it possible to run regular engines on hydrogen with a process that involves retrieving the hydrogen from water ( composed H<sub>2</sub>O).
